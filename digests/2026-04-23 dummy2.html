  • Forklift, a new chain-forking tool designed for local testing of reorgs and multiple concurrent branches on Polkadot chains, has been introduced. Forklift supports forking live chains, managing multiple branches including competing forks and reorgs, and offers helpers for relay/parachain wiring for local XCM testing. It is inspired by Chopsticks but built with PAPI and uses the @acala-network/chopsticks-executor for runtime execution. Features include YAML-based configuration, compatibility with Chopsticks’ API, and the ability to create explicit forks and storage diffs between arbitrary blocks. The tool is still in early development and currently lacks features such as persistent state and genesis bootstrapping (via 1).
