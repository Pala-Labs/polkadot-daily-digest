Polkadot Daily Digest: 2026-04-23

Technical

  • Forklift, a new chain-forking tool built with PAPI, has been introduced to address limitations in existing Polkadot testing tools by supporting multiple concurrent branches, including competing forks and reorgs. Forklift enables forking a live chain from a remote WebSocket endpoint, supports relay/parachain wiring for local XCM testing, and uses immutable Merkle-trie-backed storage with structural sharing between blocks. It is compatible with Chopsticks’ API and offers a YAML-based CLI config for multi-chain setups, though it currently lacks features like persistent state storage and genesis bootstrapping (via 1).

  • The Model Context Protocol (MCP) now allows AI tools to connect directly to Polkadot documentation, enabling agents to retrieve information as needed. MCP is compatible with platforms such as Claude AI and Codex (via 2).
